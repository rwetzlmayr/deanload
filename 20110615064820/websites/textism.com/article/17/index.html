Toward an Ad-Free Web  ::   MAR 1

It’s a remarkably simple procedure to prevent most advertisements from appearing on web pages. On the Macintosh, for example, one need merely create a text file called Hosts in the Preferences folder, containing this list of usual suspects.

Rather than requesting an ad from an ad company web server as instructed by the page, the request goes to your own computer, which goes, “Dude, I don’t think so,” and the page is all, “Dude!” and you’re all, “Sweet!”
