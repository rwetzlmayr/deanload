You Did Not Ask, But · 27 December 2001

I’ve been using Mac OSX 10.1 for a month or so, and have found it to be a most excellent playground. The BSD integration (with command line access), the multithreading and speed, the relative crashlessness of it all: it’s a very, very good operating system, especially in light of the first versions of OSX, which sucked butt.

I’m impressed by the integration of Apache web service, and, having managed just now to get PHP4 and MySQL running alongside locally (to the Macintosh enthusiast, this is like finally acquiring tickets to a game previously heard on radio only, and box seats at that), I’m feeling all generous and shit.

And so here, for those who’ve been asking after it – and for those who can make excellent use of such a thing, because after all it is all about the text – is the PHP source code for the Microsoft Word HTML Cleaner: until now only available as a no-charge service on this world-wide web site.

Best wishes to you, and yours.
