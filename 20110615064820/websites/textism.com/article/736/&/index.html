Refer 2.01 · 23 June 03

An update to everyone’s favourite vanity stockticker is available.

Those using Refer 2.0 should replace all existing files (refer.css hasn’t changed, however), fill in the preferences required by refer.php and this is important: run setupdb.php again. This won’t wipe out existing referrer data, but will simply tweak the Refer database table for improved performance, stability, robustness and suchlike hooey.

With this version, among other improvements, Refer offers a ‘glimpse’ of the top incoming site referrers recorded during the preceding 1, 2, 3 and 7 days.
