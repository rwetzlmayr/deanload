Word HTML Cleaner  ::  10 JULY 03

This utility strips proprietary Microsoft tags and artefacts from Word HTML documents, leaving basic formatting and typographic entities intact. Typographic quotes and dashes should come through fine, so long as they existed in the original Word document: run Autoformat to be sure.

This is intended for basic styled text documents; there is no support for notes, sectioning, images, &c.. Links and tables should be fine. Nonstandard tags and scripts are stripped.
