Web Lookup Scripts  ::  FEBRUARY 4

A coupla handiddlyandy Applescripts I just wrote. While similar to the Google browser toolbar script mentioned the other day, these can be run at any time from anywhere – while cheating on taxes, applying a 1-pixel feather in PhotoShop, or expanding your pornography collection.

One prompts for a Google search, the other prompts for a word to look up at Dictionary.com. I’ve been using them for an hour now, and already my quality of life has improved 3.23%.

The scripts come in two flavours, one for Internet Exploiter and iCab. If you use QuicKeys, you can either load the scripts within a macro or assign a keyboard shortcut to launch from the Finder. If you use DragThing, you can pop them onto a dock and assign a keyboard shortcut. If you use OSA Menu you can do something clever but I can’t tell you what that is because I don’t use OSA Menu.

No information about how they work OS X, because, since my Dad told me never to buy a car model in the first year of production, I’m still Classic.
