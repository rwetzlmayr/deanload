Blocking Ads  ::  MARCH 9

It’s a remarkably simple procedure to prevent most advertisements from appearing on web pages. On the Macintosh, for example, one need merely create a text file called Hosts in the Preferences Folder, containing this list of usual suspects, then restart.

Rather than sending a request to the ad company server as instructed by the page, the request goes to your own computer, which goes, “Dude, I don’t think so,” and the page is all, “Dude!” and you’re all, “Sweet!”
