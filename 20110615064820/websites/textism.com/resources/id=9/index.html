Scripted Writing for the Web  ::  FEBRUARY 4

Writing is hard. Writing web text is a pain in the ass, at least if you’re concerned about typographic standards, consistent formatting and predictable rendering of text in a variety of browsers. These scripts make it a bit less so.

What They Are · AppleScripts designed to aid in the production of text for the web.

What’s Needed to Run Them · A computer running Mac OSX or Classic MacOS, and a copy of Tex-Edit Plus, Tom Bender’s excellent shareware text editor.

How to Install · Simply copy to the Scripts subfolder inside the Tex-Edit Plus folder. It would be advised to remove any other scripts you do not plan to use, in case of possible conflicts with the keyboard shortcuts that fire the scripts.

How to Use · For the most part you can simply write as you would in any word processor (or, indeed, paste text into Tex-Edit Plus from another program), then run the Preflight Cruncher script to convert to HTML text.

A brief introduction to the scripts:

Basic Writing Scripts

Block Quote (cmd-shift-Q) will toggle the indent of the current paragraph at the left margin. When run through the Preflight Cruncher script, indented paragraphs are wrapped in <blockquote> tags.

Use Georgify (cmd-shift-G) to convert your document to 17 pt Georgia with generous leading, a good setup for writing onscreen. This script also makes HTML tags small and a different colour, making reading your text a little less noisy.

URL Grabber (cmd-shift-U) quickly makes selected text into a hyperlink pointed at whatever page is loaded in the front window of your web browser.

Use URL Wrapper (cmd-shift-H) to make a hyperlink of selected text manually.

Advanced Formatting Scripts

Block Tagger (cmd-shift-B) is useful for headings and special paragraphs that don’t require <p> tags. You need not select the entire paragraph for this to work: simply fire the script while the insertion point is in a paragraph.

Phrase Tagger (cmd-shift-T) will wrap tags around the current selected text. This is most useful for <span class=""> tags that format ranges of text within blocks.

List From Text (cmd-shift-L) will convert a range of paragraphs into a bulletted or numeric list.

Final Conversion

Here’s what Preflight Cruncher (cmd-shift-C) does:

  • Wraps HTML tags around styled text (italic, bold, underline, etc.).
  • Converts nonstandard glyphs (accented characters, analphabetic symbols and punctuation) to the coded entities more likely to render correctly in different browsers and on different operating systems.
  • Wraps <p> tags around obvious paragraphs (text followed by two carriage returns) with no existing block-level formatting tags (h1, h3, blockquote, etc.) applied.
  • Assumes text followed by a single carriage return indicates a line break within a paragraph, in which case the <br> tag is introduced.
  • Wraps <blockquote> tags around any paragraphs indented at the left margin, i.e., those on which the Block Quote script has been run.
