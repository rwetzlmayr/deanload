Refer - 17 December 02

What Is It?

Refer is a web application that tracks incoming referrers (visitors who followed a link found elsewhere) to a web site.

What Does It Do?

An up-to-the-minute list of referrers, in reverse chronological order, is displayed on a page. You can see Refer in use here.

What Is Required?

  • PHP 4.1+
  • MySQL (any recent version)
  • Your MySQL login and password must allow read/write and table creation privileges for your database.

How Can I Tell If I Have Those?

Ask your hosting provider.

Will It Work on Blogspot?

No.

What’s new in the Current Version?

With version 1.1, referrers can now be viewed in RSS form simply by appending ?format=rss to the end of the url with which you view Refer. For example: HTML view, RSS view. Thus you can now retrieve a list of current referrers in an RSS reader such as NetNewsWire Lite or Amphetadesk.

How Does One Install It?

UPGRADING TO 1.1: those who installed Refer previously need to edit and replace config.php, and replace logfuntions.php and index.php with the new versions.

  1. Download refer.zip from the link at the right and unzip it. Open the file config.php in a text editor and fill in your database information and preferences.
  2. Upload the Refer folder to the root directory of your site at your server (or replace config.php, index.php and logfunctions.php if upgrading).
  3. If installing for the first time: open a web browser and load http://yoursite/refer/setupdb.php, then follow the instructions. This checks your MySQL connection and creates the required tables in your database.
  4. On every HTML page you want monitored, paste the following:
    <?php
    $dr = $_SERVER["DOCUMENT_ROOT"];
    include ($dr."/refer/logfunctions.php");
    logit();
    ?>
  5. Once some referrers have been recorded, you’ll see them by loading http://yoursite/refer/index.php
  6. Er, that’s it.

Referrers Not Being Recorded?

Perhaps your pages are not PHP-aware. Create a text file in your web directory called .htaccess (if one exists already, open it) and paste the following inside it:
AddType application/x-httpd-php .html .htm
